# GPS Travel Calculator

This program estimates travel time based on the user's personal data.

## Features
- Automatic step length calculation based on height
- Walking time estimation
- Vehicle travel time estimation
- Output in hours and minutes format

## Usage
Run with Python 3:
    python gps_calculator.py

## Planned Features
- Elevation/slope calculation
- Weather condition integration
- Mobile health data integration (Apple Health / Google Fit)
